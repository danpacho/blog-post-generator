# Run apps

## Once

```bash
npx -y @danpacho/post-generator
```

## With scripts

1. **add to the `package.json`**

```json
{
    "scripts": {
        "post": "npx -y @danpacho/post-generator"
    }
}
```

2. **run with script**

```bash
pnpm post
```

## Installation

```bash
pnpm add -D @danpacho/post-generator
```
